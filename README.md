# NaBatata - Site de Vendas de Batatas Recheadas

Bem-vindo ao projeto **NaBatata**, um site moderno e atrativo para venda de batatas recheadas.

## Sobre o Projeto

O NaBatata é um site simples e responsivo para exibir e vender batatas recheadas com diversos sabores. O design é focado em apresentar os produtos em cards chamativos com imagens, descrições e preços, proporcionando uma experiência de compra agradável.

## Funcionalidades

- Layout responsivo para todos os dispositivos
- Cards de produtos com efeitos de hover
- Design moderno com cores quentes e atraentes
- Tipografia elegante com Google Fonts
- Integração pronta para futuras funcionalidades de e-commerce

## Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Montserrat e Poppins)

## Como Usar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em qualquer navegador
3. Os produtos podem ser facilmente editados ou adicionados no HTML

## Personalização

- Para alterar os produtos, edite o `index.html` dentro da seção `product-grid`
- Para modificar cores e estilos, edite o arquivo `styles.css`
- As imagens podem ser substituídas alterando os links no HTML
